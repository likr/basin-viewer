# SEAL-Visualizer

SI-CAT project

## Server Setup

```shell-session
$ cd server

$ npm install

$ export PGHOST=''
$ export PGUSER=''
$ export PGDATABASE=''
$ export PGPASSWORD=''
$ export PGPORT=5432

$ npm start

```

## Client Setup

Run the following command and open http://localhost:8090/ .

```
$ cd client

$ npm install

$ npm start
```

### Production Build

```shell-session
$ npm run build
```
